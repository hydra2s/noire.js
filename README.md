# 👩‍🎤 Noire.js 👩‍🎤

A new brand renderer, based on JS and Vulkan API.

### Features

- 📱 Minimalism
- 📱 Simpler
- 📱 Vulkan API 1.3.235
- 📱 RTX support
- 📱 Unified memory
- 📱 Few draw call per many models
